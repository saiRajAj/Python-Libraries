# Python Libraries Used

This project uses the following Python libraries for data analysis, visualization, and machine learning.

## Core Libraries
- **Python** – Programming language used for development
- **NumPy** – Numerical computing and array operations
- **Pandas** – Data manipulation and analysis

## Data Visualization
- **Matplotlib** – Data visualization and plotting
- **Seaborn** – Statistical data visualization

## Machine Learning
- **Scikit-learn** – Machine learning algorithms and model evaluation
  - Logistic Regression
  - Linear Regression
  - Train-test split
  - Performance metrics

## Web Application
- **Streamlit** – Interactive web application for data visualization and predictions

## Computer Vision (if applicable)
- **OpenCV** – Image and video processing
